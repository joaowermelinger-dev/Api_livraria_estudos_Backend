📚 API de Livraria

API REST desenvolvida em Node.js durante meus estudos de backend.
O projeto simula o funcionamento de uma API para gerenciamento de livros e autores, permitindo realizar operações básicas de um sistema de livraria.

Este projeto foi desenvolvido com fins educacionais, seguindo práticas de desenvolvimento de APIs utilizadas no mercado.

🚀 Tecnologias Utilizadas

- Node.js
- Express.js
- MongoDB
- Mongoose
- Nodemon

📌 Funcionalidades da API

A API permite realizar operações de CRUD (Create, Read, Update, Delete):

- 📖 Livros
- 📚 Cadastrar um novo livro
-  🔍 Buscar todos os livros
- 🔎 Buscar livro por ID
- ✏️ Atualizar informações de um livro
- ❌ Deletar um livro

🎯 Objetivo do Projeto

Este projeto foi desenvolvido para:
- Praticar Node.js e Express
- Aprender a estruturar APIs REST
- Trabalhar com MongoDB e Mongoose
- Aplicar boas práticas de organização de projeto
